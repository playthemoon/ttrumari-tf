# 뚜루마리 — 콘텐츠 진행 현황판

바이럴 TF의 롱폼/쇼츠 에피소드 진행 상황과 유튜브 채널 지표를 한눈에 보는 대시보드입니다.

---

## 1. GitHub Pages에 올리는 방법

이미 repo가 만들어져 있어요: `playthemoon/ttrumari-tf` (Public)

### Step 1. 파일 올리기
이 폴더에 있는 3개 파일을 그 repo에 그대로 올립니다.
- `index.html`
- `app.js`
- `data.js`

(저장소 페이지에서 "Add file → Upload files"로 드래그해서 올리면 됩니다.)

### Step 2. GitHub Pages 켜기
1. repo의 **Settings → Pages**로 이동
2. "Branch"를 `main`, 폴더는 `/ (root)`로 설정 후 Save
3. 1~2분 후 `https://playthemoon.github.io/ttrumari-tf/` 주소가 생성됩니다
4. 이 주소를 TF원 전체에게 공유하면 끝!

### Step 3. 데이터 안내
repo가 Public이라, 사이트 코드와 진행 상황 데이터(`data-store.json`)는 그 repo 주소를 아는 사람이라면 누구나 볼 수 있어요. 콘텐츠 기획 제목·담당자명·영상 링크 정도라 민감 정보는 아니지만, 혹시 더 비공개로 다루고 싶은 내용이 생기면 알려주세요.

---

## 2. 유튜브 구독자 수 실시간 연동하기 (선택, 최초 1회)

대시보드 안에서 **"API 키 등록하기"** 버튼을 누르면 안내가 나와요. 간단히 요약하면:

1. [Google Cloud Console](https://console.cloud.google.com/apis/library/youtube.googleapis.com) 접속
2. 프로젝트 생성 → "YouTube Data API v3" 사용 설정
3. 사용자 인증 정보 → API 키 발급
4. 대시보드에 붙여넣기

키 하나를 만들어서 TF원들에게 공유해도 되고, 각자 만들어도 됩니다. **조회 전용 API라 위험도는 낮지만**, Cloud Console에서 "HTTP 리퍼러" 제한을 걸어 본인의 GitHub Pages 주소로만 쓰이게 해두는 걸 추천해요.

## 3. 진행 상황은 어떻게 저장되나요?

**이제 TF원 전체가 같은 내용을 봅니다.** 체크박스, 영상 URL, 운영 항목, 에피소드 수정 내용 모두 `playthemoon/ttrumari-tf` repo의 `data-store.json` 파일에 저장돼요.

### 각자 한 번씩 해야 할 일 (최초 1회, 약 2분)

1. 대시보드 상단의 **"🔗 공유 저장 연결 안 됨 — 등록하기"** 클릭
2. 안내된 대로 [GitHub Fine-grained 토큰 발급 페이지](https://github.com/settings/tokens?type=beta)에서 토큰 발급
   - Repository access: 이 repo만 선택
   - Permissions → Contents → **Read and write**
3. 발급된 토큰을 붙여넣고 저장

토큰은 그 사람의 브라우저에만 저장돼요. 한 번 등록하면 그 이후로는 체크/입력할 때마다 자동으로 GitHub에 저장되고, 다른 사람이 새로고침하면 그 변경사항이 보여요.

### 동시에 여러 명이 체크하면 어떻게 되나요?

대부분의 경우 문제없이 합쳐져요 — 예를 들어 한 명이 Ep.3을 체크하는 동안 다른 사람이 Ep.7을 체크해도 둘 다 저장돼요.

다만 **정확히 같은 항목을 거의 동시에 다르게 바꾸는 경우**(예: 같은 에피소드의 같은 체크박스를 두 사람이 동시에 누름)는, 둘 중 하나만 반영될 수 있어요. 흔한 상황은 아니지만, 혹시 내가 체크한 게 새로고침 후 사라져 있다면 다시 체크해주시면 됩니다.

### 토큰 없이도 쓸 수 있나요?

네, 토큰을 등록하지 않아도 평소처럼 다 쓸 수 있어요 — 다만 그 경우엔 이 브라우저에만 저장되고 다른 사람과 공유되지 않아요.

## 4. 화면 구성

- **상단**: 유튜브 구독자/조회수/영상수 실시간 카드
- **롱폼 / 쇼츠 탭**: 기본은 **표 보기** (구글시트와 비슷한 행 나열), 카드 보기로 전환도 가능
  - 기획/이미지/영상 칸은 역할별로 색이 다르게 표시돼요 (기획=코랄, 이미지=머스타드, 영상=세이지)
  - **완료(업로드)** 칸에는 담당자가 항상 "홍신영"으로 고정 표시돼요
  - **✏️ 편집** 버튼으로 제목, 쇼츠제목, 기획/이미지/영상 담당자를 언제든 입력하거나 수정할 수 있어요. 지금 비어있는 에피소드(예: 8/26 Ep.17)도 나중에 소재나 담당자가 정해지면 바로 채워 넣을 수 있어요.
  - 체크박스 클릭으로 단계별 완료 토글
  - **조회수**: 각 에피소드 행에 "+ URL 연결"을 누르고 유튜브 영상 주소를 넣으면, 그 영상의 조회수가 자동으로 표시돼요. 나중에 URL을 채워도 그때부터 바로 연동돼요. 옆의 **▶ 버튼**을 누르면 새 탭에서 바로 그 영상으로 이동해요.
  - **운영(📌)**: 업로드, 고정댓글 등 정해진 항목이 아니라 자유롭게 할 일을 추가하는 체크리스트예요. 담당자도 적을 수 있어요.
  - **SB(📋)**: 씬별 스토리보드 연결 — 원본 시트 형식 그대로, 상단에 제작 단계(삽화/애니/촬영영상+캐릭터모션/촬영영상/더빙만) 체크와 각 씬의 구분/장면설명/오디오/자막/참고를 모두 보여줘요
- **R&R 탭**: 담당자별 역할과 주간 산출물
